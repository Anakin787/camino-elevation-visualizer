# Camino de Santiago - 고도 그래프

산티아고 순례길 1일차 구간의 고도 프로필을 시각화한 웹 그래프입니다.

## 경로 정보

- **출발지**: Saint-Jean-Pied-de-Port (생장 피에드포르)
- **도착지**: Roncesvalles (론세스바예스)
- **총 거리**: 25km
- **최고 고도**: 1,430m (Collado de Lepoeder)

## 주요 지점

1. 생장 (St-Jean) - 0km, 170m
2. 온토 (Honto) - 5.0km, 500m
3. 오리손 (Orisson) - 7.6km, 770m
4. 성모상 (Vierge) - 11.5km, 1,095m
5. 벤타르테아 (Bentartea) - 16.2km, 1,337m
6. 레포에더 (Lepoeder) - 20.5km, 1,430m
7. 론세스바예스 (Roncesvalles) - 25km, 950m

## 사용 방법

`road.html` 파일을 브라우저에서 열면 그래프를 확인할 수 있습니다.

## 기술 스택

- HTML5 Canvas
- JavaScript
- Pretendard 폰트

## 특징

- 우측에서 좌측으로 읽는 특별한 방향성
- 부드러운 베지어 곡선을 사용한 고도 표현
- 보라색/핑크 계열의 세련된 색상 디자인
- 구간별 거리 표시
- 반투명 로고 배경
